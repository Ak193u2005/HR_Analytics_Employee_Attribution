#  HR Attrition Analytics Dashboard

## Project Overview

This project analyzes employee attrition using Power BI, Python, and SQL to uncover key workforce insights.

---

##  Key Insights

* High attrition observed in **low salary segments**
* New employees show higher turnover rates
* Department-level patterns highlight risk areas
* Experience and salary strongly influence attrition

---

##  Tools Used

* Python (Pandas, NumPy, Seaborn, Matplotlib)
* SQL
* Power BI

---

##  Dashboard Features

* KPI Cards (Attrition Rate, Employees, Salary)
* Interactive Charts (Bar, Line, Donut)
* Drillthrough Pages:

  * Employee Deep Dive
  * Segment Analysis
* Custom Tooltips (3 types)
* Advanced DAX Measures
